# RFID Based Library Management System

![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)
![Technology: HTML/CSS/JS](https://img.shields.io/badge/tech-HTML%2FCSS%2FJS-brightgreen.svg)

A modern and responsive frontend interface for a library management system. This project provides a clean, dark-themed dashboard for librarians to manage books, students, and transactions efficiently. It is built entirely with vanilla HTML, CSS, and JavaScript, focusing on a clean user interface and client-side interactivity.

## Features

* **üìä Interactive Dashboard:** At-a-glance view of key statistics like total books, issued books, overdue items, and fines collected.
* **üìÑ Multi-Page Navigation:** A smooth, single-page application (SPA) feel with client-side routing to different sections: Dashboard, RFID Scanner, Books, Students, and Transactions.
* **üîç Live Table Search:** Instantly filter through books, students, and transaction records using the dynamic search functionality on each page.
* **üßæ Interactive Forms:** A dedicated "RFID Scanner" page with tabbed forms for easily issuing and returning books.
* **üé® Modern & Responsive UI:** A sleek, dark-themed design that is fully responsive and works seamlessly on both desktop and mobile devices.
* **üß© Zero Dependencies:** Built with pure, vanilla HTML, CSS, and JavaScript‚Äîno frameworks or libraries required.

## Tech Stack

* **HTML5:** For the core structure and content.
* **CSS3:** For all styling, including a responsive layout using Flexbox/Grid and custom properties for easy theming.
* **JavaScript (ES6+):** For all interactivity, including page navigation, form handling, and live search functionality.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You only need a modern web browser like Google Chrome, Firefox, or Microsoft Edge.

### Installation

1.  **Clone the repository** to your local machine:
    ```sh
    git clone [https://github.com/your-username/library-management-system.git](https://github.com/your-username/library-management-system.git)
    ```
2.  **Navigate into the project directory**:
    ```sh
    cd library-management-system
    ```
3.  **Add your screenshots** (optional):
    * Create a folder named `screenshots` in the main project directory.
    * Place your image files (e.g., `dashboard.png`, `books-page.png`) inside this folder.
4.  **Open `index.html`** in your browser. You can do this by double-clicking the file or right-clicking and selecting "Open with..." your browser of choice.
